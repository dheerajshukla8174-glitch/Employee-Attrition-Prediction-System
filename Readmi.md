# Employee-Attrition-Prediction-System
An end-to-end Employee Attrition Prediction System built using Python, Pandas, NumPy, Scikit-learn, Joblib, and Streamlit. The application analyzes key HR factors to predict whether an employee is likely to stay or leave, helping organizations make data-driven workforce decisions.
# Project Highlights
- Built an Employee Attrition Prediction system using Machine Learning.
- Compared the performance of three different algorithms:
  - Logistic Regression
  - Decision Tree
  - Random Forest
- Logistic Regression achieved the highest accuracy and was selected as the final model.
- Developed an interactive Streamlit web application for real-time predictions.
- Designed a simple and user-friendly interface for HR professionals.
